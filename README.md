# Oliver Murrell – Portfolio

A modern, dark-mode-only portfolio site built with React, Vite, Tailwind CSS, and Framer Motion. Showcases web development projects, skills, and contact information in a clean, responsive layout.

## Features

- **Project Gallery:** Interactive cards with images, summaries, and tech tags for each project
- **Skills & About Sections:** Highlights your frontend and backend skills
- **Contact Form:** Easy way for visitors to reach out
- **Dark Mode Only:** Consistent, accessible dark theme
- **Responsive Design:** Looks great on all devices
- **Animated UI:** Smooth transitions with Framer Motion

## Example Projects

- **Film & TV Series Tracker:** Track, rate, and review films and TV series with a card-based UI
- **To Do List:** Simple, persistent to-do app with PostgreSQL backend
- **Joke Generator Website:** Fetches and personalizes random jokes
- **Animal Wars:** TypeScript tournament simulation
- **Memory Game:** Simon-inspired browser game
- **TinDog:** Responsive landing page for a dog-themed dating app

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Node.js](https://nodejs.org/) (for some backend projects)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Oliver-Murrell/Portfolio.git
   cd Portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser.
